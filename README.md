# 木造耐力壁性能評価プログラム (JTCCM準拠)

**木造耐力壁及びその倍率の試験・評価業務方法書**（JTCCM）に準拠した、壁倍率算定Webアプリケーションです。

## 概要

本プログラムは、建築基準法施行令第46条第4項表1の(八)の認定申請に係る性能評価プロセスを実装しています。実験データ（正負交番載荷試験）から、以下の法的性能指標を算出します。

### 主要な評価項目
- 基礎性能値: Pmax（最大耐力）、Py（降伏耐力）、Pu（終局耐力）、μ（塑性率）
- 法的性能: P0（短期基準せん断耐力）、Pa（短期許容せん断耐力）、壁倍率

### 準拠する技術基準（要点）
- 降伏耐力Pyの算定: 作図法（第I直線[0.1-0.4Pmax]、第II直線[0.4-0.9Pmax]、第III直線[接線]の交点）
- 終局耐力Puの算定: 等価エネルギー吸収法（完全弾塑性モデル）
- 短期基準せん断耐力P0: 4つの基準 (a)降伏耐力、(b)靭性基準、(c)最大耐力基準、(d)特定変形時耐力 の最小値
- 壁倍率: Pa / (L × 1.96) で算出し、0.1刻みで端数切り捨て

## 使い方

### 1. ファイルの準備
`index.html` をブラウザで開きます。

### 2. データ入力（この版はテキスト直接入力方式）
- 変形角 γ (rad): 1行1データ
- 荷重 P (kN): 1行1データ
- ヘッダー行や空白行、単位付き文字列（例: "0.001 rad"、"3.2kN"）は自動的に無視されます。

### 3. 解析実行
「解析実行」ボタンまたは3点以上の有効データ入力で自動実行。以下が表示されます:
- グラフ: 包絡線、評価直線（Line I, III, V, VI）、特性点（Pmax, Py, Pu）
- 結果テーブル: 全性能値と壁倍率

### 4. 結果のダウンロード
- 結果CSVダウンロード: Results.csv
- グラフPNGダウンロード: Graph.png

## 注意事項

- 直接入力方式のため、CSV取込やDG1〜4等の変位計データからの自動算出機能はこの版には含まれていません。
- データは正側/負側を選択のうえ、包絡線を生成して評価します。
